1.创建一个仓库

2.仓库有默认主分支master  创建一个分支并切换 
 	git checkout -b 分支名1
	git checkout 分支名2  这一步是用来切换分支的  比如你想切换到主分支

3.在你的GitHub里面给需要合作的人开权限  设置-> 协作者  进去之后添加，等待对方同意

4.同意后随便测试一下  在他的电脑上 git init 一个文件

5.然后  git clone 你的当前仓库地址  回车

6.然后再建立一个分支git checkout -b 分支名2
   在分支名2 中上传点东西 git add. 
                        git commit 等等一系列命令上传(这些步骤都是在你新建的分支上操作的)
			
	此时  你上传的这个东西就在另外那个人的本地仓库的分支上了 
	
	然后git push到 当前网络仓库    命令为：git push 网络仓库地址

提交好之后  他的分支就提交到了网络仓库上(此时网上的仓库已经被更改，但你的本地仓库不会变)

之后你可以GitHub网上仓库查看  确定代码都无误后就可以拉到你的本地仓库并进行分支合并到你的主支

  拉下来网上仓库的代码后
  
  注意：如果你本地仓库需要合并分支  ※ 首先切换到你的本地主分支  然后通过命令 git merge 分支名 回车进行合并
